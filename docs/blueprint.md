# **App Name**: AI Sentinel

## Core Features:

- Real-time Human Detection: Utilize YOLOv8 to detect humans in the video stream with high accuracy.
- Pose Estimation: Leverage MediaPipe to estimate human poses, providing keypoint data for behavior analysis.
- Behavior Analysis: Analyze pose data to detect falls, lying motionless, and loitering, triggering alerts based on configurable thresholds. Acts as a tool.
- Alert Management: Manage alerts, providing CRUD operations with snapshot storage
- WebSocket support: WebSocket support for real-time updates
- Intuitive Dashboard: Offer a modern, dark-themed UI with real-time updates via WebSocket integration, interactive charts, and a filterable alert log, simplifying surveillance monitoring.
- Configuration Panel: Enable users to adjust camera settings, detection sensitivity, and behavior thresholds via an intuitive web UI, allowing for easy customization of the surveillance system.

## Style Guidelines:

- Primary color: Deep Indigo (#4F46E5) to convey a sense of security, intelligence, and trust.
- Background color: Dark Slate Gray (#334155) for a modern, professional, and unobtrusive feel, fitting the application's dark theme.
- Accent color: Electric Violet (#8B5CF6) to highlight key elements and actions, adding a vibrant contrast.
- Body and headline font: 'Inter', a grotesque-style sans-serif font that looks modern and neutral. Suitable for UI headlines and body text
- Code font: 'Source Code Pro' for displaying code snippets in configuration settings.
- Use clean, geometric icons from Lucide to represent different system functions, providing a consistent and intuitive user experience.
- Maintain a well-structured layout with clear visual hierarchy, utilizing white space effectively to avoid clutter and improve readability, in line with modern UI practices.
